A modern, responsive developer portfolio built with Next.js, TypeScript, and Tailwind CSS.

## Features
- **Personalized Resume Content**: About, Skills, Projects, and Contact sections with real data
- **Animated Hero Section**: Eye-catching intro with 3D/animated elements
- **Project Gallery**: Showcases featured projects with images, tech stack, and live/code links
- **Responsive Design**: Mobile-friendly and accessible
- **Custom Styling**: Tailwind CSS for fast, modern UI
- **No Next.js Branding**: Fully customized favicon and layout

## Tech Stack
- [Next.js](https://nextjs.org/) 16+
- [TypeScript](https://www.typescriptlang.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- [animejs](https://animejs.com/) (animations)
- [react-three-fiber](https://docs.pmnd.rs/react-three-fiber/getting-started/introduction) (3D elements)

## Getting Started

1. **Clone the repository**
   ```sh
   git clone https://github.com/Chandan45444/portfolio.git
   cd portfolio
   ```
2. **Install dependencies**
   ```sh
   npm install
   ```
3. **Run the development server**
   ```sh
   npm run dev
   ```
   click [Visit my website](https://portfolio-drab-theta-ka1wa17ea7.vercel.app/) to view your site.

## Customization
- Update content in the `components/` folder (About, Projects, Skills, Contact, etc.)
- Add your own images to the `app/image/` directory
- Change global styles in `app/globals.css` and Tailwind config
- Update site metadata in `app/layout.tsx`

## Deployment
Deploy easily to [Vercel](https://vercel.com/) or any platform supporting Next.js apps.
